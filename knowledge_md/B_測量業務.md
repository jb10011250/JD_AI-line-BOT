## 原則性規定QA

**原則性規定Q&A**

[**1.112年5月1日起，案件申請退費期限為幾年？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10433&type_id=10433###)

A：10年。

[**2.若土地複丈案件因未完全補正被駁回，得請求退還多少已繳規費？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10433&type_id=10433###)

A：已辦竣複丈或測量者不得退還，未辦竣複丈或測量者得請求退還（已繳規費-應繳規費之半數）。

[**3.若依法院確定判決書申請複丈，且無須實地測量者，應繳納多少規費？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10433&type_id=10433###)

A：依一般收費數額之半數計收。

[**4.若案件於複丈或測量日期前撤回，得請求退還多少已繳規費？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10433&type_id=10433###)

A：登記機關已至現場辦理整備作業者，得請求退還（已繳規費-800）元，登記機關尚未至現場辦理整備作業者，得請求退還（已繳規費-300）。

## 土地複丈QA

**土地複丈Q&A**

[**1.土地複丈費及建築改良物測量費收費標準新制何時開始實施？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：112年5月1日開始實施。

[**2.申請鑑界是否需自備制式界標？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：土地複丈施測費含土地制式界標成本，鑑界時登記機關視實地複丈情形，提供申請人土地制式界標埋設於界址點位置。

[**3.新制施行後，該如何計算申請鑑界的規費？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：可至內政部規費試算網站試算（https://easymap.land.moi.gov.tw/BSWeb/）。

[**4.想大概了解土地位置，不需釘定界標及土地複丈成果圖，可以用什麼方式呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：可以申請「位置勘查」。

[**5.位置勘查費用如何計算？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：以每筆地號為單位，每單位為500元。

[**6.土地複丈費及建築改良物測量費收費標準修正實施後，土地鑑界費用如何計算？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：鑑界費用＝基本費＋施測費；基本費共分為4種級距，分別是小於200平方公尺，基本費2500元；200以上未滿1,000平方公尺，基本費3000元；1,000以上未滿10,000平方公尺，基本費3500；10000(含)平方公尺以上，基本費4000元；施測費以每5個指定鑑定界址點為單位，每1單位以1000元計，不足5個點，以5個點計。

[**7.辦理土地分割費用如何計算？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：由基本費加上施測費，基本費以分割前每筆土地面積計算，共分為4種級距，分別是小於200平方公尺，基本費1500元；200~999平方公尺，基本費2000元；1000~9999平方公尺，基本費2500元；10000(含)平方公尺以上，基本費3000元；施測費則以分割後增加之線段為單位，每單位500元。

[**8.申請界址調整有限制嗎？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：土地界址曲折調整者，應檢附界址曲折調整協議書，並以同一地段、地界相連、使用性質相同之土地可達成減少地界線為限。如為實施建築管理地區，並應符合建築基地法定空地分割辦法之規定。若土地設有他項權利，應先徵得他項權利人之同意。

[**9.若我只需要測量4個界址點位，可以申請退還施測費嗎？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

 A：施測費以5個點為單位，不足5個點以5個點計算，故無退費之適用。

[**10.可以於現場需要增加施測界址點嗎？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###)

A：若增加後施測界址點的數量未超出原申請施測單位，且相關鄰地人有到場，可於現場直接施測，倘超出原申請施測單位或有未事先通知之鄰地關係人，則需依地籍測量實施規則第212條規定辦理補正，修改土地複丈申請書後，另行安排時間測量。

[**11.**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10402&type_id=10402###) **鑑界現場需要注意什麼？**

A：為確保鑑界作業順利進行並保障申請人權益，請留意以下：

一、清除障礙物：請於排定測量日前，預先清除現場可能阻礙測量視線或動線之障礙物。若測量員到場時因障礙物致無法實施測量，將依規定開立補正通知書；申請人須於接到通知書之日起十五日內補正完畢，逾期未補正將依法予以駁回。

二、核對身分：鑑界當天，請原申請人或所有權人本人親自到場，並攜帶「身分證明文件正本」，以供測量員核對身分。

三、委託代理：若本人無法親自到場，委由代理人出席時，請代理人務必攜帶「代理人身分證正本」及「委託書」。

四、準備工具與埋設界標：本所測量員會於現場提供制式界標（包含界樁及鋼釘），惟仍請申請人自行準備鐵鎚或噴漆等工具，以便於確認界址後，由申請人自行將制式界標釘立埋設。

**12.下雨天會照常進行測量嗎？**

A：若遇雨天或其他氣候因素，將由測量員依據現場天候狀況評估是否實施複丈。詳細說明如下：

一、當日主動洽詢：因天氣變化難以精準預測，建議申請人(鄰地關係人)於鑑界當天一早先行致電本所洽詢該案測量員，確認當日是否如期進行測量作業。

二、測量作業評估：下雨天除實地作業困難外，亦會影響儀器觀測之測量精度，且易造成精密設備損壞。為確保測量品質與成果正確性，若雨勢過大致無法實施測量，測量員將決定暫緩作業。

三、改期與書面通知：若因風雨或其他事故致當日不能實施複丈，本所將依規定辦理改期。考量地所未留存無鄰地關係人之聯絡電話，致無法即時以電話聯繫，後續將一律以「書面通知」申請人及關係人改期複丈之時間，尚祈見諒。

13土地鑑界申請人資格﹕

一﹑凡土地經界不明者﹐由土地所有權人或管理人申請之。

二﹑因承租土地經界不明者﹐應由承租人會同土地所有權人或管理人申請。

三﹑共有土地得由部分共有人單獨申請之。

四﹑公有土地同意他人建築使用﹐如能檢具確實證明文件﹐得由使用人代位申請辦理。

五﹑繼承人檢具被繼承人(原土地所有權人)死亡後戶籍謄本(除戶證明)﹐得以證明其繼承人之身份﹐並於土地複丈申請書備註欄加註無拋棄繼承權情事者﹐得由繼承人代位申請辦理。

## 土地複丈流程表

|  |  |
| --- | --- |
| 作業流程 | 步驟說明 |
| 受理申請 | 申請人填具土地複丈及標示變更登記申請書(或土地複丈申請書),檢附身分證明文件向地政事務所申請 |
| 收件 | 壹、收件人員應核對申請人或代理人身分無誤後,將收件號數、土地坐落、申請人等資料鍵入電腦予以收件。貳、收件人員計徵規費。參、申請人繳納規費。 |
| 分配案件 | 壹、收件後隨即將電腦自動配件予承辦測量人員。 |
| 排定日期 | 參、電腦自動排定測量日期,列印土地複丈案件收據由申請人收執。肆、依內政部訂頒土地複丈費及建築改良物測量費標準核計規費,填寫於複丈費欄。伍、收費人員依前點規費金額列印地政規費及其他收入收據。陸、測量案件交由測量員簽收。 |
| 審查 | 一、依「土地法」、「地籍測量實施規則」、「數值地籍測量土地複丈作業手冊」、「辦理圖解法土地界址鑑定作業注意事項」、「辦理土地複丈與建物測量補充規定」、「圖解法地籍圖數值化成果辦理土地複丈作業手冊」、「土地複丈費及建築改良物測量費標準」、「地籍資料電子處理系統規範」等相關法令規定予以審查。二、應提文件、申請人(代理人)資格、土地複丈申請書格式及填載事項、有關規費是否符合規定。三、對申請之不動產標示、權屬與地籍資料所載是否相符。 |
| 通知 | 複丈案件經審查無誤後,寄發通知單予申請人及鄰地關係人。 |
| 補正通知 | 地政事務所受理複丈申請案件,經審查有「地籍測量實施規則」規定之下列各款情形者,應通知申請人於接到土地複丈案件補正通知書之日起15日內補正。一、申請人之資格不符或其代理人之代理權有欠缺者。二、申請書或應提出之文件與規定不符者。三、申請書記載之申請原因與登記簿冊或其證明文件不符,而未能證明不符原因者。四、未依規定繳納土地複丈費者。 |
| 補正完畢 | 補正完畢後再排定測量日期。 |
| 駁回通知 | 地政事務所受理複丈申請案件,收件後經審查有「地籍測量實施規則」規定之下列情形者,應以駁回通知書敘明法令依據或理由駁回之。一、不屬受理地政事務所管轄者。二、依法不應受理者。三、逾期未補正或未依補正事項完全補正者。四、實地複丈遇障礙物無法辦理,並經公文通知限期排除逾期未排除者予以駁回。 |
| 核對地籍圖及原有地籍圖、查表調製土地複丈圖 | 壹、圖解法複丈依「地籍測量實施規則」規定,依地籍圖或圖解地籍圖數化成果調製土地複丈圖時,應將其鄰接四周適當範圍內之經界線及附近圖根點,精密移繪或繪製於圖紙上,並應將界線之彎曲、鄰接圖廓線及圖面折皺損等情形繪明之。數值法複丈以電腦展繪方式調製土地複丈參考圖後核對原有地籍調查表及原有土地複丈圖,並核算面積是否合於「地籍測量實施規則」之規定,並將查對結果填載於辦理鑑界案件作業自我檢查紀錄表。貳、經查對原有圖冊、面積資料發現有不符情形應查明其不符原因後,依「地籍測量實施規則」有關規定辦理釐正或更正。參、辦理複丈前應先檢查測量儀器,校正是否符合規定,並應作定期之儀器檢查與維護。 |
| 前置 | 壹、依複丈日期、時間及會同地點到場,依「地籍測量實施規則」規定先查核申請人身分及關係人所執土地複丈定期通知書是否相符。關係人如未到場者,得逕行複丈。若申請人或代理人因故無法到場,得委託他人代理,但應檢附委託書。 |
| 申請人認章 | 貳、複丈結束後,測量人員應在複丈圖上註明界標名稱、界址點編號及其關係位置,並依「地籍測量實施規則」規定,請申請人在土地複丈圖或土地複丈參考圖上簽名或蓋章。參、土地複丈成果圖倘申請人於現場拒絕領取,須於土地鑑界辦理完竣後,另函文寄送之。 |
| 成果整理 | 壹、依「地籍測量實施規則」規定整理土地複丈圖。貳、整理外業手簿資料,如檢測紀錄、戶地測量光線法觀測手簿並填寫土地界址鑑定作業紀錄表。參、面積計算:數值法複丈以界址點坐標計算之,圖解法複丈以實量距離、圖上量距、坐標讀取儀或電子求積儀測算之,並填寫土地面積計算表。 |
| 成果檢查、成果核定 | 壹、成果整理後,測量員應先自行檢查,再移送檢查人員及課長,依「地籍測量實施規則」有關規定檢查、核定成果。貳、經檢查有誤退回原測量員重新成果整理。 |
| 結案歸檔 | 土地複丈申請書併其附件全宗、土地複丈圖及土地複丈成果圖按序歸檔結案。 |
| 通知申請人領狀 / 訂正地籍圖冊 | 複丈案件為土地分割或合併案件,測量課結案後會移至登記課辦理登記,登記完成會通知申請領取新繕權狀。土地複丈分割及合併案件土地標示部確定已分割完成,請圖庫人員訂正相關地籍圖冊。 |

## 建物測量QA

**建物測量Q&A**

**[1.請問申辦建築管理後之建物第一次測量或複丈應檢附哪些資料呢？](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413" \l "##" \o "請問申辦建築管理後之建物第一次測量或複丈應檢附哪些資料呢？)**

[**2.請問繪製建物測量成果圖或建物標示圖需使用什麼繪圖軟體呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：開業之建築師、測量技師、地政士或其他與測量相關專門職業及技術人員轉繪建物測量成果圖或繪製簽證之建物標示圖，請以內政部建物測量繪圖軟體（SBpublic）製作。

A：倘僅附紙本成果而未附規定電子檔者，申請人應另繳納每建號600元之建物平面圖及位置圖數值化作業費，並由登記機關製作電子檔。

[**4.請問依地籍測量實施規則第282-2條規定辦理建物第一次測量所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：新式建物測量收費標準新增轉繪簽章成果圖核對費額，每建號200元。

[**5.請問申辦建物合併所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物合併複丈費每單位為400元（每建號每層每五十平方公尺為一單位，不足五十平方公尺者，以五十平方公尺計），並以合併前每建號之面積計算。以轉繪者，轉繪費每單位為400元（每建號每層每五十平方公尺為一單位，不足五十平方公尺者，以五十平方公尺計）。

[**6.請問申辦建物分割所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物分割複丈費每單位為1000元（每建號每層每五十平方公尺為一單位，不足五十平方公尺者，以五十平方公尺計），並以分割後每建號之面積計算。以轉繪者，轉繪費每單位為800元（每建號每層每五十平方公尺為一單位，不足五十平方公尺者，以五十平方公尺計）。

[**7.請問申辦建物部分滅失所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物部分滅失測量費每單位為1000元（以滅失後存餘建物之面積計算，每建號每層每五十平方公尺為一單位，不足五十平方公尺者，以五十平方公尺計），以轉繪者，轉繪費每單位為800元（以每建號為一單位）。

[**8.請問申辦建物基地號或建物門牌號勘查所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物基地號或建物門牌號勘查費為每建號500元。

[**9.請問申辦建物全部滅失所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物或特別建物各棟次之全部滅失勘查費為每建號500元。

[**10.請問申辦建築管理後之建物第一次測量「轉繪建物位置圖」，需要檢附什麼資料呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物位置圖申請以轉繪方式辦理者，應檢附經實地測繪且由開業之建築師或測量技師或其他依法規得為測量相關簽證之專門職業及技術人員簽證之建物地籍測繪資料。未檢附上開建物地籍測繪資料者，登記機關應請申請人補正檢附之，或請申請人修正申請書並補繳規費（每棟新臺幣4,000元）後，由登記機關排定日期辦理實地測量建物位置，據以製作建物測量成果圖。

[**11.請問附屬建物之「陽臺」是否可以保存呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：附屬建物「陽臺」之認定，除使用執照竣工圖說應標註「陽臺」外，並應審認是否屬依據建築技術規則建築設計施工編第3款及第162條第1項第1款規定尺寸及面積檢討設置之「陽臺」。

[**12.請問依地籍測量實施規則第282條規定辦理建物第一次測量所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物位置圖測量費為整棟建物4000元及建物平面圖測量費每單位為1000元（每建號每層每五十平方公尺為一單位，不足五十平方公尺者，以五十平方公尺計）。

[**13.請問依地籍測量實施規則第282-1條規定辦理建物第一次測量所需繳納之規費是多少呢？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10413&type_id=10413###)

A：建物位置圖轉繪費為每建號200元及建物平面圖轉繪費為每建號800元。

| 排定日期 | 壹、電腦自動排定測量日期後,列印案件收據由申請人收執。貳、依內政部訂頒土地複丈費及建築改良物測量費標準核計規費,填寫於複丈費欄。參、收費人員依前點規費金額列印地政規費及其他收入收據。肆、測量案件交由測量員簽收。 |
| 通知 | 電腦自動排定後隨即通知測量日期及時間。 |
| 審查 | 壹、依「土地法」、「地籍測量實施規則」、「辦理土地複丈與建物測量補充規定」、「土地複丈費及建築改良物測量費標準」、「簡化建物第一次測量作業要點」、「建物所有權第一次登記法令補充規定」等相關法令規定予以審查。貳、核對應提文件,申請人資格、代理人權限、建物測量申請書格式及填載事項、有關規費是否合乎規定。參、核對申請之不動產標示、權屬與地籍資料所載是否相符。肆、案件經審查有誤即通知補正。 |
| 補正通知 | 經審查有「地籍測量實施規則」規定之下列各款情形者應通知申請人於接到地政事務所建物測量案件補正通知書之日起15日內補正。一、申請人之資格不符或其代理人之代理權有欠缺者。二、申請書或應提出之文件與規定不符者。三、申請書記載之申請原因與登記簿冊或其證明文件不符,而未能證明不符原因者。四、未依規定繳納建物測量費者。 |
| 補正完畢 | 補正完竣後再排定測量日期。 |
| 駁回通知 | 經審查有「地籍測量實施規則」規定之下列情形者,應以駁回通知書敘明法令依據或理由駁回之。一、不屬受理地政事務所管轄者。二、依法不應受理者。三、逾期未補正或未依補正事項完全補正者。 |
| 查對地籍圖、土地、建物登記簿及原建物測量成果圖 / 實地勘測 / 駁回通知 / 成果整理(填寫勘查結果) | 調製建物測量圖 一、依「地籍測量實施規則」規定,調製建物測量圖後,應核對地籍圖、原有建物測量圖後,始得辦理測量。二、辦理建物測量前應先檢查測量儀器,校正是否符合規定,並應作定期之儀器檢查與維護。壹、實施測量時,應依測量日期、時間及會同地點到場,依「地籍測量實施規則」規定先查核申請人身分及所執地政事務所建物測量定期通知書是否相符。若申請人或代理人因故無法到場,臨時委託他人代理者,應檢附委託書。貳、原定測量日期,因風雨或其他事故不能測量時,依「地籍測量實施規則」規定,經簽報核准改期後,通知申請人。參、申請人屆時不到場,依「地籍測量實施規則」規定,不予測量,視為放棄建物測量之申請,已繳建物測量費不予退還。但申請人撤回申請者,不在此限。肆、圖解區採平板儀或數值區採經緯儀測量,依「地籍測量實施規則」規定,測量建物位置。伍、測量完畢後,依「地籍測量實施規則」規定,請申請人在建物測量圖上簽名或蓋章。經現場勘查發現線場建物因增改建致與原建物保存資料不相符時,應以公文通知限期檢附相符之合法建物證明文件,倘無法補正應予以駁回。依「地籍測量實施規則」規定製作建物測量成果圖,包括繪製建物位置圖及建物平面圖或建物勘查應依實地勘查結果,填寫於地政事務所建築改良物勘查結果通知書。 |
| 成果檢查 | 成果整理後,測量員應先自行檢查,再移送檢查人員及課長,檢查有誤退回原測量員重新整理成果。 |
| 核定 | 依「地籍測量實施規則」有關規定核定成果。 |
| 核發勘查結果 | 檢查核定後,由地政事務所發給建築改良物勘查結果通知書。 |
| 通知申請人領結果通知 | 通知申請人領取由地政事務所發給建築改良物勘查結果通知書。 |
| 建物複丈申請書歸檔 | 建物測量申請書併其附件全宗、測量成果圖按序歸檔。 |

## 配套措施QA

**配套措施Q&A**

**[1.請問申辦建物第一次測量應如何繳交符合規定之共通格式電子檔？](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10427&type_id=10427" \l "##" \o "請問申辦建物第一次測量應如何繳交符合規定之共通格式電子檔？)**

A：可使用免費建物測量繪圖軟體（SBpublic），該軟體提供完整建物測量成果圖繪製功能，並可匯入DXF 向量格式及上傳成果資料。

[**2.請問如何取得免費建物測量繪圖軟體（SBpublic）？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10427&type_id=10427###)

A：可至內政部地政司網站（網址easymap.land.moi.gov.tw/K01/）下載。

[**3.請問如何取得免費土地試分割軟體（SBpublic）？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10427&type_id=10427###)

A：可至內政部地政司網站（網址easymap.land.moi.gov.tw/L01B/）下載。

[**4.請問有土地想要分割，該如何規劃分割方案？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10427&type_id=10427###)

A：可以使用免費土地試分割軟體（SBpublic），土地所有權人可透過自然人憑證申請地籍圖作業檔，載入試分割系統執行土地分割試算，並可輸出試分割作業成果報表及參考圖。

[**5.收費標準修正後，我該如何事先預估申請鑑界需要多少複丈規費？**](https://sinhu.land.hsinchu.gov.tw/content/?parent_id=10427&type_id=10427###)

A：可運用「土地複丈規費服務網」（easymap.land.moi.gov.tw/BSWeb/）提供之線上規費估算服務，進行土地鑑界基本費及施測費估算、土地分割基本費估算，估算結果亦提供下載，可自行列印與申請書一併檢附送件。
